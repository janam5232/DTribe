This repository contains a python script specifically created to extract excel data and upload it to a database.

##Technologies Used:
1. Python
2. AWS EC2
3. AWS RDS

####Requirements:
1. Python 3.9.2
2. Python Libraries:
  - openpyxl
  - pandas
  - mysql.connector

####OS:
It has been tested on Windows 10, 8 and Linux Ubuntu and CentOS.
